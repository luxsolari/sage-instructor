description: "Create a new curriculum interactively"
---
Read the Sage instructor skill at `.claude/skills/sage-instructor/SKILL.md` and execute the `/new-track` command. Run the Track Setup flow from the Onboarding section, skipping Round 0 — the learner already expressed custom-track intent by running this command. Use AskUserQuestion to interview the learner about what they want to learn, their destination project, current level, and priority. Generate a curriculum file from the template and save it.
